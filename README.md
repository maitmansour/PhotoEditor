## Functionalities
- [x] Select a directory
- [x] List all pictures on directory
- [x] Choose a picture
- [x] Update a Picture Tags 
- [x] Internationalization (Ar/Fr/En)
- [ ] Crop / Rotate / Slide
- [x] Find by Keyword

## Technologies
- NetBeans
- Scene Builder
- JavaFX
- JavaFX Concurrent Task
